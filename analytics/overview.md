# Analytics Overview

Learn how to track and analyze your link performance in LinkUTM.

## Overview

LinkUTM provides comprehensive analytics to help you understand how your links perform, where your traffic comes from, and which campaigns drive the best results. Track every click, analyze user behavior, and make data-driven decisions.

## Analytics Dashboard

### Key Metrics

**Total Clicks**

- Goal completions tracked
- Conversion rate

- ROI calculations

**Click-Through Rate (CTR)**
- Percentage of impressions that resulted in clicks

- CTR by channel

### Connect GA4

LinkUTM integrates with Google Analytics 4:

1. **OAuth Connection:**

   - Authorize LinkUTM
   - Grant necessary permissions

2. **Select Account:**
   - Choose GA4 account
   - Select property
   - Choose data stream

3. **Data Sync:**
   - Automatic data synchronization
   - Real-time or near-real-time updates
   - Historical data access

4. **View in LinkUTM:**
   - GA4 metrics in LinkUTM dashboard
   - Combined view with LinkUTM analytics

## Common Analytics Questions

### How many clicks did my campaign get?

1. Go to **Links** or **Analytics**
2. Filter by campaign (utm_campaign)
3. View total clicks
4. Compare to goals

### Which channel drives the most traffic?

1. Go to **Analytics**
2. View **By Source** or **By Medium**
3. Sort by total clicks
4. Analyze top performers

### What's my best-performing link?

1. Go to **Links** page
2. Sort by **Total Clicks** or **Conversions**
3. View top link
4. Analyze what makes it successful

### Where is my traffic coming from?

1. View link analytics
2. Check **Geographic Data** section
3. See country, state, city breakdown
4. Review referrer sources

### What devices do my users use?

1. View link analytics
2. Check **Device Information**
3. See desktop vs. mobile split
4. Review OS and browser data

## Analytics Limitations

### Data Privacy

**Privacy Compliance:**
- GDPR compliant
- Privacy-first tracking
- Anonymized data where required
- User consent respected

**What's Not Tracked:**
- Personal identifying information
- Email addresses
- Names
- Private browsing sessions (limited data)

### Data Accuracy

**Factors Affecting Accuracy:**
- Ad blockers (may block tracking)
- VPNs (affect geographic data)
- Bot traffic (filtered automatically)
- Browser privacy settings
- Cached pages

**Best Practices:**
- View trends over time
- Focus on relative performance
- Compare campaigns fairly
- Use multiple data sources

### Data Retention

**Storage Periods:**
- Raw click data: Check your plan
- Aggregated data: Longer retention
- Archived campaigns: May have limits
- Export for long-term storage

## Troubleshooting

### No Data Showing

**Possible Issues:**
- Link just created (wait a few minutes)
- No clicks yet
- Tracking disabled
- Ad blocker interference
- Analytics integration inactive

**Solutions:**
- Wait 5-10 minutes for data
- Test link yourself
- Check link settings
- Verify analytics connection
- Contact support

### Data Discrepancies

**Why Numbers May Not Match:**
- Different tracking methods
- Attribution windows
- Bot filtering
- Deduplication logic
- Time zone differences

**Solutions:**
- Use same date range
- Check timezone settings
- Understand each platform's methodology
- Focus on trends, not exact numbers

### Geographic Data Missing

**Possible Causes:**
- VPN usage
- Privacy settings
- Limited data

**Solutions:**
- View at aggregate level
- Focus on available data
- Combine with other metrics
