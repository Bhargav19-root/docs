# Folders

Learn how to organize your links using folders in LinkUTM for better campaign management.

## Overview

Folders help you organize your links into logical groups, making it easy to find, manage, and analyze links for specific campaigns, clients, or time periods. With folders, you can keep your workspace clean and your links well-organized.

## Why Use Folders?

### Benefits

**Better Organization**
- Group related links together
- Separate campaigns or projects
- Keep workspace clean and organized

**Easier Navigation**
- Quickly find specific campaign links
- Filter by folder
- Reduce clutter

**Team Collaboration**
- Clear project structure
- Easy handoffs between team members
- Organized workspace for everyone

**Better Analytics**
- View metrics by folder
- Compare folder performance
- Track campaign groups

## Creating Folders

### Method 1: During Link Creation

1. When creating a new link, find the **Folder** dropdown
2. Click **Create New Folder**
3. Enter folder name
4. Click **Save** or **Create**
5. The folder is now available for this and future links

### Method 2: From Folders Page

1. Navigate to **Folders** in the sidebar (if available)
2. Click **Create Folder** or **+ New Folder**
3. Enter folder details:
   - **Name**: Descriptive folder name
   - **Description**: Optional folder description (if supported)
4. Click **Create**

### Method 3: From Links Page

1. Go to your **Links** page
2. Look for folder management options
3. Click **Add Folder** or similar option
4. Name your folder
5. Save

## Managing Folders

### Viewing Folder Contents

1. Go to **Links** page
2. Click the **Folder** dropdown or filter
3. Select a folder to view its links
4. See all links within that folder

### Default Folder

Set a default folder for new links:

1. Go to folder settings
2. Mark a folder as **Default**
3. All new links automatically assigned to this folder
4. Can be changed during link creation

**Benefits of Default Folder:**
- Streamlines link creation
- Ensures no links are unorganized
- Can be changed per-link as needed

### Moving Links Between Folders

**Single Link:**
1. Find the link to move
2. Click **Edit** on the link
3. Change the **Folder** dropdown
4. Click **Save**

**Multiple Links:**
1. Select multiple links (if bulk actions supported)
2. Click **Bulk Actions** or **Move**
3. Choose destination folder
4. Confirm move

### Renaming Folders

1. Navigate to folder settings
2. Find the folder to rename
3. Click **Edit** or **Rename**
4. Enter new name
5. Save changes

**Note:** Renaming doesn't affect links or analytics

### Deleting Folders

#### Empty Folders

1. Ensure folder has no links
2. Navigate to folder settings
3. Click **Delete** or trash icon
4. Confirm deletion

#### Folders with Links

**Option 1: Move Links First**
1. Move all links to another folder
2. Delete the now-empty folder

**Option 2: Delete with Reassignment**
1. Delete the folder
2. System prompts to reassign links
3. Choose a destination folder
4. Confirm deletion

**Warning:** Some systems may delete links with folder. Always check before confirming!

## Folder Features

### Folder Statistics

View key metrics for each folder:
- **Total Links**: Number of links in folder
- **Total Clicks**: Aggregate clicks across all links
- **Active Links**: Links currently active
- **Created Date**: When folder was created

### Folder Search

Find folders quickly:
1. Use the search bar
2. Type folder name
3. Filter results

### Folder Filtering

Filter your links page by folder:
1. Click **Folder** filter
2. Select one or more folders
3. View only links from those folders

### Folder Sorting

Sort folders by:
- **Name** (A-Z or Z-A)
- **Date Created** (Newest or Oldest)
- **Link Count** (Most to Least)
- **Recent Activity** (Most Recent)

## Troubleshooting

### Can't Find a Folder

**Solutions:**
- Use folder search
- Check if folder was deleted
- Verify you have access permissions
- Ensure you're in the correct project

### Links Not in Expected Folder

**Solutions:**
- Check link's folder assignment
- Move link to correct folder
- Verify default folder setting
- Review recent edits in activity log

### Folder Deleted Accidentally

**Solutions:**
- Check if undo is available (immediately after deletion)
- Contact support for recovery (if available)
- Links may have been moved to default folder
- Check folder archive (if available)

### Can't Delete Folder

**Possible Reasons:**
- Folder contains links (move or delete them first)
- Folder is set as default (change default first)
- Insufficient permissions
- System default folder (cannot be deleted)
