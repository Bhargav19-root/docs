# UTM Templates

UTM Templates allow you to create reusable parameter sets for consistent campaign tracking across your marketing initiatives.

## Creating Templates

1. Navigate to the Templates section
2. Click "New Template"
3. Define your default parameters:
   - Template name
   - Default source
   - Default medium
   - Default campaign structure
4. Save your template

## Template Features

### Variable Parameters
- Use placeholders for dynamic values
- Support for custom variables
- Bulk URL generation with variable substitution

### Template Management
- Organize templates by category
- Share templates with team members
- Template versioning
- Template duplication

### Template Rules
- Parameter validation
- Required field enforcement
- Character limitations
- Naming conventions

## Using Templates

1. Select a template
2. Fill in any variable parameters
3. Generate your UTM URL
4. Save or export the generated URL

## Template Best Practices

- Create templates for different marketing channels
- Maintain consistent naming across templates
- Document template usage guidelines
- Regularly review and update templates
- Archive unused templates
