# UTM Parameters Guide

UTM parameters are tags added to URLs to track the effectiveness of your marketing campaigns in analytics tools.

## Standard UTM Parameters

### Required Parameters

1. **utm_source**
   - Identifies the source of your traffic
   - Examples: google, facebook, newsletter
   - Required for all tagged URLs

2. **utm_medium**
   - Identifies the marketing medium
   - Examples: cpc, email, social
   - Required for all tagged URLs

3. **utm_campaign**
   - Identifies the specific campaign
   - Examples: spring_sale, product_launch
   - Required for all tagged URLs

### Optional Parameters

4. **utm_term**
   - Used for paid search keywords
   - Optional parameter
   - Examples: running_shoes, blue_widgets

5. **utm_content**
   - Used for A/B testing or content-specific tracking
   - Optional parameter
   - Examples: blue_button, header_link

## Parameter Guidelines

### Naming Conventions
- Use lowercase letters
- Use underscores or hyphens instead of spaces
- Keep names short but descriptive
- Be consistent across campaigns

### Common Values

#### Source (utm_source)
- google
- facebook
- twitter
- linkedin
- newsletter
- email

#### Medium (utm_medium)
- cpc
- email
- social
- display
- banner
- referral

## Best Practices

1. **Consistency is Key**
   - Use standardized naming conventions
   - Document your parameter strategy
   - Share guidelines with team members

2. **Keep it Simple**
   - Avoid unnecessary parameters
   - Use clear, descriptive names
   - Don't overcomplicate tracking

3. **Regular Maintenance**
   - Review parameter usage
   - Clean up unused parameters
   - Update naming conventions as needed
