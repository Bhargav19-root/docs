# API Authentication

Learn how to authenticate your requests to the UTM Builder API.

## Authentication Methods

### API Keys
```bash
# Header Authentication
curl -H "Authorization: Bearer YOUR_API_KEY" \
  https://api.utmbuilder.com/v1/links

# Query Parameter
curl https://api.utmbuilder.com/v1/links?api_key=YOUR_API_KEY
```

### OAuth 2.0
```bash
# OAuth Token
curl -H "Authorization: Bearer YOUR_OAUTH_TOKEN" \
  https://api.utmbuilder.com/v1/links
```

### Code Examples

```python
import requests

headers = {
    'Authorization': 'Bearer YOUR_API_KEY'
}

response = requests.get(
    'https://api.utmbuilder.com/v1/links',
    headers=headers
)
```

```javascript
fetch('https://api.utmbuilder.com/v1/links', {
    headers: {
        'Authorization': 'Bearer YOUR_API_KEY'
    }
})
.then(response => response.json())
.then(data => console.log(data));
```
