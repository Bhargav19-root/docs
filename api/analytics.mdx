# Analytics API

API endpoints for retrieving and analyzing link performance data.

## Endpoints

### Get Link Analytics
```bash
GET /v1/analytics/links/{link_id}
```

### Get Campaign Analytics
```bash
GET /v1/analytics/campaigns/{campaign_id}
```

### Get Team Analytics
```bash
GET /v1/analytics/team
```

## Query Parameters

### Time Range
```bash
GET /v1/analytics/links/{link_id}?start_date=2024-01-01&end_date=2024-01-31
```

### Metrics
```bash
GET /v1/analytics/links/{link_id}?metrics=clicks,conversions,revenue
```

### Grouping
```bash
GET /v1/analytics/links/{link_id}?group_by=day&timezone=UTC
```

## Response Format

### Link Analytics
```json
{
  "link_id": "link_123",
  "time_range": {
    "start": "2024-01-01T00:00:00Z",
    "end": "2024-01-31T23:59:59Z"
  },
  "metrics": {
    "clicks": 1000,
    "unique_clicks": 800,
    "conversions": 50,
    "revenue": 1500.00
  },
  "timeline": [{
    "date": "2024-01-01",
    "clicks": 100,
    "unique_clicks": 80
  }]
}
```

### Campaign Analytics
```json
{
  "campaign_id": "camp_123",
  "metrics": {
    "total_links": 10,
    "total_clicks": 5000,
    "conversion_rate": 2.5
  },
  "links": [{
    "link_id": "link_123",
    "clicks": 1000
  }]
}
```
