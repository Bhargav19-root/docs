# Links API

API endpoints for creating and managing UTM links.

## Endpoints

### Create Link
```bash
POST /v1/links

{
  "url": "https://example.com",
  "utm_source": "facebook",
  "utm_medium": "cpc",
  "utm_campaign": "spring_sale"
}
```

### Get Link
```bash
GET /v1/links/{link_id}
```

### Update Link
```bash
PATCH /v1/links/{link_id}

{
  "utm_campaign": "summer_sale"
}
```

### Delete Link
```bash
DELETE /v1/links/{link_id}
```

## Query Parameters

### List Links
```bash
GET /v1/links?page=1&limit=20&sort=created_at
```

### Filter Links
```bash
GET /v1/links?utm_source=facebook&created_after=2024-01-01
```

## Response Format

### Single Link
```json
{
  "id": "link_123",
  "url": "https://example.com",
  "short_url": "utmb.io/abc123",
  "utm_params": {
    "source": "facebook",
    "medium": "cpc",
    "campaign": "spring_sale"
  },
  "created_at": "2024-01-01T00:00:00Z",
  "updated_at": "2024-01-01T00:00:00Z"
}
```

### Link List
```json
{
  "data": [{
    "id": "link_123",
    "url": "https://example.com",
    // ... link data
  }],
  "meta": {
    "page": 1,
    "total": 100,
    "limit": 20
  }
}
```
