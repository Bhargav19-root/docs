# Welcome to UTM Builder Docs

A lightweight, fast, and reliable UTM tracking + link shortener platform for marketing teams, agencies, and developers.

Use this documentation to:

* Create short links with UTM parameters
* Connect custom domains
* Automate with API & Zapier
* Track campaign performance with real-time analytics
