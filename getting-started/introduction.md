## Key Concepts

### UTM Parameters

**What They Are:**
Tags added to URLs to track marketing campaigns in analytics tools.

**Five Standard Parameters:**
- `utm_source`: Where traffic comes from (facebook, google, newsletter)
- `utm_medium`: Type of channel (social, email, cpc)
- `utm_campaign`: Specific campaign name (summer_sale_2024)
- `utm_term`: Keywords or audience (optional)
- `utm_content`: Differentiates similar links (optional)

**Example URL:**
```
https://yoursite.com/page?utm_source=facebook&utm_medium=cpc&utm_campaign=summer_sale&utm_content=carousel_ad
```

[Learn more about UTM parameters ‚Üí](/utm-parameters/overview)

### Link Shortening

**What It Is:**
Converting long URLs with UTM parameters into short, clean links.

**Example:**
```
Long URL: https://example.com/product?utm_source=facebook&utm_medium=social&utm_campaign=launch
Short URL: https://yourbrand.link/abc123
```

**Benefits:**
- Professional appearance
- Easier to share
- Better click-through rates
- All tracking preserved

[Learn more about link shortening ‚Üí](/link-shortening/overview)

## Getting Started

### Quick Start (5 Minutes)

1. Go to linkutm website
2. Click **Sign Up**
3. Enter email and create password
4. Verify email

**Step 2: Complete Onboarding**
1. Enter workspace name (your company)

## Common Use Cases

### Use Case 1: Social Media Campaign

**Scenario:** Running Facebook ad campaign for product launch

**Setup:**
1. Create UTM template:
   - utm_source: `facebook`
   - utm_medium: `cpc`
   - utm_campaign: `product_launch_q2`
2. Create links for each ad variant
3. Use utm_content to differentiate: `video_ad`, `carousel_ad`, `image_ad`
4. Enable link shortening
5. Use short links in Facebook ads

**Result:**
- Track which ad creative performs best
- Measure ROI per ad
- Optimize budget allocation

### Use Case 2: Email Newsletter

**Scenario:** Weekly newsletter with multiple CTAs

**Setup:**
1. Create newsletter template:
   - utm_source: `newsletter`
   - utm_medium: `email`
   - utm_campaign: `weekly_digest_[date]`
2. Create separate link for each CTA
3. Use utm_content: `featured_article`, `product_cta`, `event_registration`
4. Add to appropriate folder
5. Tag with üü° Yellow (Email)

**Result:**
- Track which newsletter sections get most clicks
- Understand subscriber interests
- Optimize newsletter layout

### Use Case 3: Multi-Channel Product Launch

**Scenario:** Launching new product across email, social, and paid search

**Setup:**
1. Create "Product Launch Q2" folder
2. Create template with utm_campaign: `product_launch_q2`

   - Email: source=newsletter, medium=email

4. Set up UTM rules for consistency

## FAQs

### Do I need technical knowledge?

No! linkutm is designed for marketers, not developers. If you can create a social media post, you can use linkutm.

### Will this work with my analytics tool?

Yes! UTM parameters are standard and work with Google Analytics, Adobe Analytics, Mixpanel, Amplitude, and most analytics platforms.

### Can I use this with my team?

Absolutely! Invite unlimited team members (depending on your plan), assign roles, and collaborate seamlessly.

### Do I need to own a domain for link shortening?

No, you can use linkutm's default shortener. Custom branded domains are optional but recommended for customer-facing links.

### What happens if I cancel?

Your links continue to work (if you used custom domain you own), but you lose access to the dashboard and analytics. Always export your data first.

### Is there a limit on links?

Depends on your plan. Most plans offer generous or unlimited link creation.

### Can I import existing links?

Yes! Use CSV import to bulk import existing links with UTM parameters.

### How long is data retained?

Check your plan details. Most plans offer 1-2 years of detailed analytics, with aggregated data retained longer.
